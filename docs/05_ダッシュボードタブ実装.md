## 実装メモ
- `st.columns` でグラフを横並び配置
  ```python
  col1, col2 = st.columns(2)
  with col1:
      st.plotly_chart(fig1, use_container_width=True)
  with col2:
      st.plotly_chart(fig2, use_container_width=True)
  ```
- フィルター変更時は自動でグラフが更新されるようにする
